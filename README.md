# AI Brain — Persistent Artificial Intelligence 🧠

AI Brain is an experimental, next-generation AI research initiative designed to explore **persistent memory systems**, **autonomous agents**, and **long-term cognitive architectures**. Unlike traditional AI models that treat every interaction as a fresh start, AI Brain is built with an integrated localized knowledge graph that allows it to remember, learn, and evolve.

## 🚀 Key Features

- **Next-Gen AGI Search Console:** A high-speed, terminal-style search interface that simulates cognitive boot sequences and parses intention before execution.
- **Persistent AI Memory:** Uses a localized JSON-based knowledge graph to store user facts, relationship mapping, and long-term topics.
- **Multi-Provider AI Router:** Native support for **Gemini 1.5**, **OpenAI GPT-4o**, **Claude 3**, and **Custom Provider** endpoints with automated self-healing fallback logic.
- **Agent Frameworks:** Specialized personality modes for different domains (Teacher, Startup Advisor, Coding Assistant).
- **Premium UI/UX:** A glassmorphic, high-performance interface with real-time 3D visualizations (Three.js), network graphs, and interactive particles.
- **Privacy First:** API keys are stored only in your browser's local storage and never sent to our servers.

## 🛠️ Technology Stack

- **Frontend:** HTML5, Vanilla JavaScript (ESM), CSS3 (Modern variables + Grid/Flexbox).
- **Core Engine:** Vite.js for high-speed builds and development.
- **Graphics:** [Three.js](https://threejs.org/) for the cognitive brain visualization and interactive 3D elements.
- **AI Integrations:** 
  - Google Generative AI (Gemini)
  - OpenAI API
  - Anthropic (Claude)
  - Open AI compatible Custom Endpoints

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/bsjat8110/ai-brain-website.git
   cd ai-brain-website
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## ⚙️ Configuration

No server-side environment variables are required. Once the app is running:
1. Open the **AI Brain Chat** via the search terminal or hotkey (`⌘K` / `Ctrl+K`).
2. Click the **Settings (⚙️)** icon.
3. Choose your preferred AI Provider and enter your **API Key**.
4. The key is saved locally in your browser for future sessions.

## 🔬 Research Areas

- **Topic Persistence:** How AI can maintain context across months of intermittent conversation.
- **Autonomous Planning:** Developing internal models for agents to plan multi-step actions.
- **Cognitive Mapping:** Visualizing how the AI's internal knowledge graph evolves in real-time.
